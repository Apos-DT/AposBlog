01 / Layout

整体布局

三栏 CSS Grid 布局,左 240px · 中 minmax(0, 1fr) · 右 280px;主区使用 min-width: 0 避免内容溢出;两侧用 position: sticky 固定;宽度切换用 vw 与 calc()。

240px 左侧目录 sticky · 章节锚点
1fr · min 0 主内容区 grid 内可滚动 · 不溢出
280px 工具面板 sticky · 调色 / 字号
布局策略:外层 display: grid + 三栏列宽,< 1100px 自动塌成单列。
